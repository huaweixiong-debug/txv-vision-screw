# 膨胀阀拧紧防错追溯 HMI 使用说明书

## 一、启动系统

双击 `D:\expansion_valve_hmi\启动HMI.bat`，等待浏览器自动打开 `http://192.168.0.99:8010`。

## 二、界面概览

页面顶部有四个标签页：**生产** | **设置** | **数据** | **训练数据**。

## 三、生产操作

### 选择产品和操作员
- **产品型号**：下拉选择（HB11 / E22H / E211）
- **操作员**：下拉选择操作员姓名
- 切换产品会自动写入奇力速拧紧枪参数

### 自动流程（7 步）
1. **O 型圈检测** — 放置产品，相机自动识别 2 个 O 型圈
2. **膨胀阀检测** — 放上膨胀阀，检测到后自动发送 PLC M0.0
3. **压紧** — PLC 就绪（M0.5）后自动进入拧紧
4. **拧紧** — 逐个拧紧两颗螺丝，NG 不记录需重拧，OK 自动记录
5. **判定** — 全部 OK 发 PLC M0.1，开始扫码
6. **扫码** — 扫码枪自动触发，扫描二维码绑定产品
7. **完成** — 发 PLC M0.2，自动开始下一轮

### PLC 信号说明
| 信号 | 方向 | 含义 |
|------|------|------|
| M0.0 | PC→PLC | 产品就绪 |
| M0.1 | PC→PLC | 拧紧完成且全部 OK |
| M0.2 | PC→PLC | 扫码完成 |
| M0.3 | PLC→PC | 手动模式（=1 不自动启动） |
| M0.4 | PLC→PC | 急停 |
| M0.5 | PLC→PC | PLC 就绪 |
| M0.6 | PLC→PC | 复位（回到初始状态） |
| M1.0 | PC→PLC | 拧紧 NG |

## 四、数据查询

1. 切换到「数据」标签页
2. 选择 **开始日期** 和 **结束日期**
3. 可选输入关键字（序列号 / 二维码 / 操作员）
4. 可选筛选状态
5. 点击「查询」
6. 点击「导出 Excel」导出为 Excel 文件

Excel 中的图片路径可点击打开。

## 五、数据记录说明

- **序号格式**：`产品型号-日期-工位-序号`（例：`HB11-20260521-ST01-0001`）
- 每天每个型号序号从 0001 开始
- 仅拧紧 OK 时分配正式序号
- NG 螺丝不记录，需重新拧紧直至 OK
- 时间记录为两颗螺丝全部拧紧完成的时刻

## 六、导出目录

- Excel 日报：`D:\膨胀阀装配数据\{月份}\{日期}.xlsx`
- 抓拍图片：`D:\膨胀阀装配数据\images\{型号}\{日期}\`
- 运行日志：`D:\expansion_valve_hmi\runtime\hmi_stdout.log`
